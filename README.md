# Phishing-Detector

A Python-based phishing detection tool that analyzes URLs using blacklist detection and suspicious pattern analysis.

## Features
- Blacklist detection
- HTTPS checking
- URL keyword analysis
- IP address detection
- URL shortener detection
- Risk scoring system

## Technologies
- Python
- Regex
- URL parsing

## How to Run

```bash
python3 detector.py
```

## Example

Input:
http://fake-login.com/verify-account

Output:
High Risk Phishing URL
